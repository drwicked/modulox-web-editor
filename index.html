Patches are pre-programmed parameters for specific synths or applications. You must connect a number of modules as indicated in brackets.
Mix allows you to control a DJ mixer app. Find it here

The presets apply to any type of configuration. For example, he can put Toggles on the entire first line by incrementing the MIDI notes (1,2,3 ...).
The patches load a specific configuration that you have saved. You need exactly the same number of connected modules as for the saved configuration.
To make things easier for you, you can load a preset that is close to what you want to do, then modify the parameters as you wish, and save the patch thus created.

When you connect a USB device, it appears in the list. To connect to the modulox, select STM32 AUDIO USB, then click on Connect.
The modules then appear in the interface and you can configure them. To save the configuration in the module, click on the Write button

. Toggle : ON / OFF mode. When the button is ON, it takes on the color defined for Press1, and the MIDI CC signal is sent when the potentiometer is turned. When the button is OFF, it takes on the color defined for Press2, and the MIDI CC signal is no longer sent when the encoder is turned.
Two different MIDI notes can be defined when switching ON or OFF. These are the notes for Press1 and Press2.
. Momentary : Push-button mode: when the button is pressed, a MIDI note (Press) is sent. And when it is released, a MIDI note (Release) is also sent. You can send the NOTE OFF signal of the note sent by pushing the button, or any other note (useful for DAWs).
. Selector : Multi-switch mode, giving several states to the button. For example, when you click once note 1 is sent, and the button turns green, a second time it's note 12 and the button is red, a third time note 5 and the button is yellow. The number next to the mode allows you to choose how many different states you want to be able to have. Note that the DC signal sent by turning the knob is always the same in this mode.
. Select CC : Same principle as the previous mode, we can have several states. But when you change the mode, no MIDI note is sent. On the other hand, the DC signal sent when the button is turned is different for each state.
