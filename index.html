Les patchs sont des paramètres préprogrammés pour des synthés ou des applications précises. Il faut connecter un nombre de module comme indiqué entre parenthèses.
Mix permet de contrôler une appli de mixage de DJ. Retrouvez-là ici

Les presets s'appliquent à tout type de configuration. Par exemple il peut mettre des Toggle sur toute la première ligne en incrémentant les notes MIDI (1,2,3...).
Les patchs chargent une configuration bien précise que vous avez sauvegardée. Il faut exactement le même nombre de modules branchés que pour la configuration sauvegardée.
Pour vous simplifier la tâche, vous pouvez charger un preset se rapprochant de ce que vous voulez faire, puis modifier les paramètres à votre guise, et sauvegarder le patch ainsi créé.

Quand vous connectez un appareil USB, il apparaît dans la liste. Pour se connecter au modulox il faut sélectionner STM32 AUDIO USB, puis cliquer sur Connecter.
Les modules apparaissent alors dans l'interface et vous pouvez les configurer. Pour sauvegarder la configuration dans le module, il faut cliquer sur le bouton Write

. Toggle : C'est un mode ON/OFF. Quand le bouton est sur ON, il prend la couleur définie pour le Press1, et le signal MIDI CC est envoyé quand on tourne le potentiomètre. Quand le bouton est OFF, il prend la couleur définie pour Press2, et le signal MIDI CC n'est plus envoyé quand on tourne le potentiomètre.
Deux notes MIDI différentes peuvent être définies quand on passe en ON ou OFF. Ce sont les notes pour Press1 et Press2.
. Momentary : C'est un mode bouton-poussoir : quand on appuye sur le bouton, une note MIDI (Press) est envoyée. Et quand on le relâche, une note MIDI (Release) est aussi envoyée. Vous pouvez envoyer le signal NOTE OFF de la note envoyée en poussant le bouton, ou tout autre note (utile pour les DAW).
. Selector : C'est un mode multi-switch, donnant plusieurs états au bouton. Par exemple, quand vous cliquez une fois la note 1 est envoyée, et le bouton devient vert, une deuxième fois c'est la note 12 et le bouton est rouge, une troisième fois la note 5 et le bouton est jaune. Le nombre à côté du mode permet de choisir combien d'états différents on veut pouvoir avoir. A noter que le signal CC envoyé en tournant le bouton est toujours le même dans ce mode.
. Select CC : Même principe que le précédent mode, on peut avoir plusieurs états. Mais quand on change le mode, aucune note MIDI n'est envoyée. Par contre le signal CC envoyé à la rotation du bouton est différent pour chaque état.
